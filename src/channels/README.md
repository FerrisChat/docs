# Channels

A Channel object is structured as follows:

| Name     | Type    | Description                       |
|----------|---------|-----------------------------------|
| id       | u128    | channel id                        |
| name     | String  | channel name                      |
| guild_id | u128    | id of the guild the channel is in |
